# Chennai House Price Prediction

This project is a **machine learning application** to predict house prices in Chennai, India. It uses historical real estate data and various property features to train a model that estimates prices accurately.

## Features

- Predicts house prices based on input features such as location, area, number of bedrooms, and other property characteristics.
- Supports data cleaning and preprocessing for accurate predictions.
- Provides a ready-to-use model for local deployment or further improvements.
